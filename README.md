# Low Utility Sequence Mining (LUSM)

This project implements the Low Utility Sequence Mining (LUSM) algorithm based on the research paper "Efficient Mining of Low-Utility Sequential Patterns". It includes both naive and optimized approaches.

## Features
- Naive implementation
- Optimized implementation
- Memory usage tracking

## Project Structure
- `LUSPM/` : Main project folder
- `src/` : Java source code
- `graphs/` : Output graphs
- `resources/` : Dataset files

## How to Run
1. Compile Java files
2. Run `MainLUSMAlgorithm.java`
3. Provide dataset input

## Dataset
As per in the research paper

## Base Paper
Efficient Mining of Low-Utility Sequential Patterns
